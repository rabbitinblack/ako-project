# Ako Project

Ako Project is a bilingual Thai and Japanese arithmetic flashcard app for simple child practice.

## Language

**Flashcard**:
A practice card that shows one arithmetic problem on the front and its answer on the back.
_Avoid_: Quiz item, question card

**Addition Deck**:
The set of flashcards for addition problems.
_Avoid_: Plus mode, add card set

**Subtraction Deck**:
The set of flashcards for subtraction problems that starts with 0 through 10 minus 0, then continues with positive-answer subtraction up to 10 minus 9.
_Avoid_: Minus mode, subtract card set
